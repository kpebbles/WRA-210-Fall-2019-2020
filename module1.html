Module 1: Understanding Servers

What You'll Learn
This module will help you gain a basic understanding of the Internet, specifically about servers, hosts, file transfers, and your MSU-provided Netfiles.

What You'll Produce
You will produce a .pdf in which you answer a series of questions about the readings. You will then upload that document to your Netfiles.

Step-by-Step Instructions
  1. Read / watch the following:
    • What is a Web Host?
    • What is a Web Host
    • What is FTP?
    • MSU Netfiles
  2. Open a Microsoft Word document in which you answer the following questions:
    • What are the 3 basic technologies needed to run a website?
    • What are local files, what are web servers, what is FTP and how are they related?
    • What is MSU Netfiles and what does it allow you to do?
    • What is your personal URL for the web space MSU provides for you through MSU Netfiles?
  3. Save your Word document as a .pdf. Name your file module01.pdf - no spaces, no special characters (!@#$%^&*). When you've saved the file, be sure to use netfiles.msu.edu to upload it to your web folder so it'll be accessible on the Internet. After the file has completed uploading, make sure you can access the file using a web browser. The link should look like http://msu.edu/~YOURNETID/module01.pdf
  4. To submit Module 1: edit the link for Module 1 on your course page to the link above.
Additional Helpful Information

Creating Your First Page

Download the personal course site template, saving it as “index.html”.

Download a coding program (read about what coding programs are available here , but I recommend Brackets or Visual Code Studio...do not plan on using Notepad or Notepad ++ for the rest of the semester).

Open the personal course site template using your new coding program by right clicking your “index.html” file and selecting “Open with…” and then the name of your program (e.g., Brackets).

Using your coding software, modify just the written text on your index.html file page (i.e., your name, and your about me description). Post this “index.html” page to your netfiles space (link), using the instructions availablehere.

Make sure to name your file is named exactly“index.html” (do not use capital letters, and make sure you save it as .html not .htm).

Check the link to your student sitehereto make sure that you’ve correctly uploaded your page to your Netfiles file space.

If it doesn’t work, make sure that you’ve uploaded it into the “Web” folder, check the naming (including that it’s an “.html” file). Make sure you don’t have any capital letters for special characters. Your file should simply be “index.html” file. Email me (or direct message me in Slack) if you have difficulties.
